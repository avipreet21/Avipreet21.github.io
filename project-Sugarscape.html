Sugarscape Model

This project utilizes coding techniques to analyze a dataset spanning from 1990 to 2019, focusing on deaths attributed to drug disorders across various countries

CODE LINK
Project Overview The presented code implements a simplified Sugarscape model, an agent-based simulation designed by Epstein and Axtell to study societal phenomena, particularly wealth distribution. The model consists of an artificial environment where agents move, consume, and collect sugar based on individual attributes like metabolism and vision. The environment regenerates sugar, allowing for the observation of emergent patterns. The code provides classes for Sugarscape and agents, offering a platform to explore scenarios related to economic dynamics, resource management, policy analysis, urban planning, and educational simulations. This model serves as a versatile tool for understanding complex socio-economic systems and their real-world implications.
