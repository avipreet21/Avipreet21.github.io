Grazing Model

This computational framework specifically targets the simulation of grazing systems in a 2D Cellular Automata context, providing insights into the dynamics of ecosystem interactions.

CODE LINK
Project Overview The project implements a 2D Cellular Automata model simulating a grazing system where deer interact with grass in a self-regulating manner. Utilizing Python libraries such as NumPy and Matplotlib, the model tracks the state of the grazing land, deer attributes, and grass growth, allowing for the exploration of various parameters and initial conditions. Through the GrazingCA and Deer classes, the simulation iterates through steps to update deer behavior, including movement towards nearby grass patches, weight adjustments, and survival checks. The Animation2D class enables visualization of the simulation, facilitating analysis of the system's stability and the impact of different factors on the deer population and grass availability.
