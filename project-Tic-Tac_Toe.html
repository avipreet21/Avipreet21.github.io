Tic-Tac-Toe

The Tic-Tac-Toe project is a console-based game implemented in C that includes a basic artificial intelligence (AI) component.

CODE LINK
Project Overview The project is a C program that implements a Tic-Tac-Toe game where a player can play against the computer. The program begins with a brief explanation of the game rules and allows the user to play multiple games if desired. The game logic is implemented using functions to initialize the board, display the board, handle the player's turn, handle the computer's turn, check for a winner, and check if the board is full. While the computer's move is randomly selected in this implementation, the project lays the foundation for understanding game development and basic artificial intelligence concepts in programming.
