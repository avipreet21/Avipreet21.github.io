DRUG RELATED DEATHS ANALYSIS

This project utilizes coding techniques to analyze a dataset spanning from 1990 to 2019, focusing on deaths attributed to drug disorders across various countries

CODE LINK
Project Overview This project utilizes Python for analyzing drug-related deaths, employing descriptive statistics, logistic regression, and regression analysis. It focuses on understanding demographic disparities in drug use disorder rates among various age groups, genders, and ethnicities to tailor prevention and intervention strategies. Additionally, it investigates correlations between drug use disorders and socio-economic factors like income levels, education, employment rates, and urbanization to inform targeted interventions. Moreover, the project explores pharmaceutical trends and prescription patterns, particularly the relationship between opioid prescription rates, availability, and opioid use disorder prevalence. Through data analysis, the project aims to provide insights for addressing drug-related issues effectively.
